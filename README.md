# Tymofii's Portfolio

Static multi-page portfolio designed for GitHub Pages deployment.

## Files

- `index.html` home page
- `about.html` about page
- `projects.html` portfolio/projects page
- `resume.html` recruiter-focused resume page
- `contact.html` contact page with static form UX
- `404.html` custom not-found page
- `styles.css` shared visual system
- `script.js` theme toggle, mobile nav, reveal effects, and form placeholder behavior

## Personalize Before Deploy

1. Replace placeholder testimonials with real references.
2. Optionally add a PDF resume export alongside the included `.docx`.
3. Keep expanding the projects page as you ship more work.

## Deploy

Upload the contents of this folder to a GitHub Pages branch or repository root.
